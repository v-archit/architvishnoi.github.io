First Person Shooter
Android

Basic Movement

It is a Prototype of First Person Shooter Android Game with the objective to kill the terrorists in a given time. State Automata is employed for animation switching with input. Screen is divided into parts for joystick movement and screen rotation. Screen-to-Ray mechanism is used to rotate the player when swiping.
Tags: Unity3D-C#-FPS-Android
Role: Lead Developer
Skills: Character Animation, State Automata, UI Design, UI Input, Raycasting

Button Interaction

Player can also change his stride to running or walking using a toggle button. There is also an option to switch to scoped gun mode for better visibility. Bullets are fired using Fire button.
Tags: Unity3D-C#-FPS-Android
Role: Lead Developer
Skills: Character Animation, State Automata, UI Design, UI Input, Raycasting

Gun Fire / Kill Animation

Bullets are guided using raycasting with a slight dip over distance. Enemy AI follows pre-defined Navmesh paths and gets trigerred when in vicinity. There is random noise applied to Enemy gun for difficulty manipulation. Ragdoll animation is used when an enemy/player dies.
Tags: Unity3D-C#-FPS-Android
Role: Lead Developer
Skills: Character Animation, State Automata, UI Design, UI Input, Raycasting
