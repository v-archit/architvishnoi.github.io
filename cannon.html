Cannon Blaster

3D Cannon Ball Shooter
Multiplayer. Windows

Server Side

It is a 3D Cannon Shooter Windows Game with the objective to strike the enemy player with cannon ball. This is a Multiplayer version using UNET Networking to sync data between client and server. 3D Rigidbody force is used to launch the ball with an explosion particle system. Balls are illuminates with Point light. Floor has low friction for an entertaining experience.
Tags: Unity3D-C#-3D-Networking-Multiplayer-Windows
Role: Solo Project
Skills: Game Logic, UI Design, Collisions, Rigidbody Force, Networking, Particle System, Lighting

Client Side

Only 2 instances of game is allowed and it should be on the same local network. There are 2 types of cannon ball. Red ones have high damage and passes through walls. Green ones have low gamage and gets reflected back from the walls for 10 seconds. Arrow keys are used for input.
Tags: Unity3D-C#-3D-Networking-Multiplayer-Windows
Role: Solo Project
Skills: Game Logic, UI Design, Collisions, Rigidbody Force, Networking, Particle System, Lighting
