Diamond Robber

2D Platformer
Windows

Basic Level

It is a 2D Platformer Windows Game with the objective to collect diamonds in unlimited time and stay on the platforms while jumping. 2D Rigidbody force is used to make the player jump. Trigger functions are used to check if the player falls. Points are earned for collecting a diamond.
Tags: Unity3D-C#-2D-Indefinite Platformer-Windows
Role: Lead Developer
Skills: Game Logic, UI Design, Trigger Handling, 2D Rigidbody Force, Sprite Animation

Advanced Level

Advanced level changes to full fledged game with 3 lives to increase difficulty. It changes to a timed game with 2 different types of diamonds for more points. If a player collides with Police sprite then the points are deducted. Diamonds keep on switching with police for strategical difficulty.
Tags: Unity3D-C#-2D-Indefinite Platformer-Windows
Role: Lead Developer
Skills: Game Logic, UI Design, Trigger Handling, 2D Rigidbody Force, Sprite Animation
