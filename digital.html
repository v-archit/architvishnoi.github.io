Digital Gandhi Gyan-Vigyan

It is an Augmented Reality app which augments information about personal items like Letters etc. that are displayed in the exhibition. It uses Vuforia Model Tracking and Image Tracking functionality to augment digital model of Utensils and 2D pictures of letters/maps on the displayed items. User can gain knowledge about the lifestyle of Mahatma Gandhi using this immersive experience.
Tags: Unity3D-C#-Augmented Reality-Vuforia-Android-eJourney
Role: Lead Developer, UI Designer
Skills: Vuforia Model/Image Recognition, UI Logic, Optimization

Digital Gandhi Akar-Prakar

It is an Augmented Reality app which showcases the life journey and important quotes of Mahatma Gandhi. It uses Vuforia Model Tracking functionality to augment digital model of Gandhiji's signature over the 3D printed Vuforia-Recognized sculpture. User can choose between AR and Non-AR modes of this app. A voiceover is incoporated with every piece of information displayed.
Tags: Unity3D-C#-Augmented Reality-Vuforia-Android-eLearning
Role: Lead Developer, UI Designer
Skills: Vuforia Model Recognition, UI Logic, Voiceover Integration, Optimization

VRchive Sabarmati Ashram

Virtual Reality Application built using SteamVR API on Unity Engine. It is running on HTC Vive with 2 Base Stations and Wireless Controllers. It is a Treasure Hunt game where user needs to find hidden objects that are emitting sound. Player can hold these objects using controller.
Tags: Unity3D-C#-Virtual Reality-SteamVR-HTC Vive
Role: Lead Developer
Skills: Gameplay Mechanics, Game Logic, Level Design, Optimization, Lighting, Navmesh AI, Rigidbody Joints, VR Controller Mapping

ViRaasat Sabarmati Ashram

Mixed Reality Installation consisting of a to-scale 3D printed model of Sabarmati Ashram. It consists of Projection Mapping and an interactive virtual walk-through of the digitally modeled monument placed in Unity3D enviroment.
Tags: Unity3D-C#-3D Printing-Projection Mapping
Role: Level Designer
Skills: Building Placement, Optimization, Navmesh AI
